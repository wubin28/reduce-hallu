# 减少 AI 幻觉 — 页面设计文档

**日期：** 2026-05-25  
**状态：** 已批准

---

## 概述

将 Next.js Hello World 页面替换为「减少 AI 幻觉」工具页面。该工具通过自动为用户输入的提示词添加「减幻前缀/后缀」，帮助用户得到更可信的 AI 回答。

---

## 功能规格

### 核心行为

1. 用户在多行文本框中输入待优化的提示词
2. 点击「查询事实」按钮后：
   - 自动在提示词前添加前缀：`你是专家。`
   - 自动在提示词后添加后缀：`请提供主要观点的3个不同出处的网页链接以便我查验。如果你不知道或查不到，就实说，不要编造。`
   - 清空输入框
   - 以**打字机动画效果**逐字符显示优化后的提示词（在输入框上方的结果区）
3. 结果区支持点击一键复制

## 视觉设计

- **主题色：** 橙色系（`#f97316` 主橙，`#ea580c` 深橙，`#fed7aa` 浅橙边框，`#fff7f0` 橙色背景）
- **页头：** `linear-gradient(135deg, #ea580c, #f97316, #fb923c)` 渐变背景，白色文字
- **按钮：** 渐变橙色，带阴影悬浮效果
- **结果区：** 白色卡片，橙色左边框，橙色标签
- **响应式：** 使用 Tailwind CSS，移动端垂直堆叠，桌面端输入框与按钮横排

## 技术方案

### 文件变更

- `app/page.tsx` — 替换为全新客户端组件（`"use client"`）

### 关键实现细节

- **状态管理：** `useState`（输入框内容、输出文本、是否输出中）
- **打字机效果：** `useEffect` + `setTimeout` 逐字符追加到显示字符串
- **结果区显隐：** 有结果文本时才渲染结果卡片
- **复制功能：** `navigator.clipboard.writeText()`
- **响应式：** Tailwind `flex-col sm:flex-row` 处理输入区布局

### 减幻提示词模板

```
前缀：你是专家。
后缀：请提供主要观点的3个不同出处的网页链接以便我查验。如果你不知道或查不到，就实说，不要编造。
```
